## Extensions

To run this project, you will need to add the following extensions for vs code.

`https://marketplace.visualstudio.com/items?itemName=mike-lischke.vscode-antlr4`

Used settings:

```
"antlr4.generation": {
        "language": "CSharp",
        "mode": "external",
        "listeners": false,
        "visitors": true,
        "outputDir": "./ANTLRGenerated/"
}
```
